Working with Git and Github :

1. Create a Github Profiel and Repository for the project
2. Download Git in your system and install it
3. Open terminal and Type "git" to check the installation status 
4. ![image](https://github.com/user-attachments/assets/3667369e-0456-498d-950e-a092e851ed00)
5. Configure Git Username and Email in terminal by using below codes
 git config --global user.name "Your Username"
 git config --global user.email "your-email@example.com"
